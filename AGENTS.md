# AGENTS

## Overview
This repository hosts **OptiUniverse**, an iOS 3D universe navigator that focuses on extremely beautiful rendering of celestial objects using MetalKit.

## Development Guidelines
- Use Swift and MetalKit for all rendering and UI work.
- Prioritise visual fidelity and performance when rendering 3D objects.
- Keep code clean, documented and modular.

## Testing
- Ensure the project builds before committing:
  `xcodebuild build`
- Run unit tests when available:
  `xcodebuild test`
