Suggested Shader Parameters for Corona/Rim Glow

coronaIntensity: 1.0 – base multiplier for emissive strength
coronaScale: 0.6 – controls radial gradient scale (match to corona size)
flickerSpeed: 1.2 – UV scroll speed for noise (can vary per preset)

Usage:
  vec2 uv = ...; // radial coords
  float grad = texture(coronaGradient, uv).r;
  float flicker = texture(coronaNoise, uv*coronaScale + time*flickerSpeed).r;
  float corona = grad * (0.8 + 0.2*flicker);
  emissive += corona * coronaIntensity;
